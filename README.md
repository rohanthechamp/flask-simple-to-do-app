# 📝 Flask To-Do App

A simple To-Do web application built using Flask to manage daily tasks.

---

## 🚀 Features

- Add tasks  
- Mark tasks as completed  
- Delete tasks  
- Set priority (0–10)  
- View all tasks  

---

## 🛠️ Tech Stack

- Flask  
- SQLAlchemy  
- Flask-WTF  
- SQLite  
- Bootstrap  

---

## ⚙️ Setup

```bash
git clone https://github.com/rohanthechamp/flask-simple-to-do-app.git
cd flask-simple-to-do-app

python -m venv venv

python App.py
